# Coffee Sales Performance Dashboard

## Business Question
What drives coffee shop revenue, and how can sales performance be improved?

##  Problem Solved
This Excel dashboard helps coffee shop management:
- Identify top and low-performing items  
- Analyze revenue contribution by product type and category  
- Measure average customer transaction value  
- Track sales trends over time  
- Compare revenue performance across multiple locations  

##  Key Insights & Benefits
- Prioritize high-performing items to ensure stock availability  
- Optimize, bundle, or remove low-performing items  
- Enable data-driven inventory and promotional decisions  
- Provide management with a clear, consolidated view of sales performance  

##  Tools Used
- Microsoft Excel (Pivot Tables, Charts, Formulas)


## Conclusion
This dashboard empowers coffee shop stakeholders to make informed decisions that optimize sales performance and increase profitability through data-driven strategies.
